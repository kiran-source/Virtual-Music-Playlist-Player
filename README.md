# 🎵 Virtual Music Playlist Player

A modern web-based music player built using **HTML, CSS, and JavaScript** that demonstrates the practical implementation of the **Doubly Linked List** data structure. The application allows users to create and manage multiple playlists, upload local audio files, navigate between songs, shuffle music, and enjoy a real-time audio waveform visualizer.

## ✨ Features

* 🎶 Play local audio files
* 📂 Create and manage multiple playlists
* ➕ Upload songs to any playlist
* ⏭️ Next and ⏮️ Previous song navigation
* 🔀 Shuffle songs randomly
* 🗑️ Delete songs from playlists
* 🎼 Doubly Linked List implementation for efficient song traversal
* 📊 Real-time audio waveform visualizer using the Web Audio API
* 🌈 Multiple UI themes with one-click theme switching
* 📱 Responsive and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Web Audio API
* HTML5 Audio Element
* HTML5 Canvas

## 📚 Data Structure Used

### Doubly Linked List

Each playlist is implemented using a **Doubly Linked List**, where every song is represented as a node containing:

* Song name
* Audio file
* Previous song pointer
* Next song pointer

This enables:

* Efficient forward and backward navigation
* Easy insertion of new songs
* Quick deletion of songs
* Smooth playlist traversal

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/kiran-source/Virtual-Music-Playlist-Player.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.

No additional installation or dependencies are required.

## 🎧 How to Use

1. Open the application.
2. Create a new playlist or choose an existing one.
3. Upload your local music files.
4. Click a song to start playing.
5. Use the Previous, Next, and Shuffle controls.
6. Switch between themes for a customized experience.
7. Delete songs whenever needed.
